## Eggheadz & Friends Consulting
This page serves as the documentation for Eggheadz & Friends Consulting. As your products continue to evolve so will this page. 

#### Running the Project
```bash
# clone repository
git clone <repo> && cd <repo>
# install deps
npm install
# run dev server
npm start
# build optimized code & run build
npm run build && npm run start:build
```

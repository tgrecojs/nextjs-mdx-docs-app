### CSS-in-JS 
These UI components leverage the [emotion-js](emotion.sh) CSS-in-JS library. Throughout the code you will come across components that use the object-literals for defining styles. 

```.jsx
<button style={{
    background: '#b495ed',
    color: '#fff',
    fontSize: '16px',
    padding: '8px',
    border: '2px solid #26666d',
    borderRadius: '4px',
    textAlign: 'center'
}}>
    Come on... gimme a lil click! :)
</button>
```
